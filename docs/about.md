<div class="max-w-2xl mx-auto">
  <h2 class="text-2xl font-bold mb-6">关于 AI Daily Digest</h2>
  
  <div class="prose prose-invert max-w-none">
    <p class="text-gray-300 mb-6">
      AI Daily Digest 是一个自动化的AI资讯日报系统。每天早8点，系统会自动从多个可靠信源抓取最新的AI领域资讯，经过智能筛选和整理后，生成精美的静态网页。
    </p>
    
    <h3 class="text-xl font-semibold mb-4 text-gray-200">📊 内容特色</h3>
    <ul class="space-y-2 text-gray-300 mb-6">
      <li>• 每日精选 5-10 条高质量AI资讯</li>
      <li>• 涵盖研究进展、行业动态、工具资源</li>
      <li>• 多源验证，确保信息准确可溯源</li>
      <li>• 智能评分，优先展示重要内容</li>
    </ul>
    
    <h3 class="text-xl font-semibold mb-4 text-gray-200">🤖 技术架构</h3>
    <ul class="space-y-2 text-gray-300 mb-6">
      <li>• 抓取：Python + RSS/API</li>
      <li>• 站点：Jekyll + GitHub Pages</li>
      <li>• 样式：Tailwind CSS</li>
      <li>• 自动化：GitHub Actions</li>
    </ul>
